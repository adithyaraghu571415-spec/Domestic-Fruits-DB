# Fruits Dashboard

A business analytics dashboard for tracking domestic fruit sales, margins, and performance metrics.

## Features
- Real-time data synchronization with Firebase
- CSV data integration from Google Sheets
- Sales and margin tracking by owner
- Source market performance analysis
- Trade corridor analytics

## Deploy to Vercel

1. Install Vercel CLI (if not already installed):
   ```
   npm i -g vercel
   ```

2. Login to Vercel:
   ```
   vercel login
   ```

3. Deploy from this directory:
   ```
   vercel
   ```

4. For production deployment:
   ```
   vercel --prod
   ```

## Alternative: Deploy via GitHub
1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com)
3. Import the repository
4. Vercel will auto-detect and deploy the static site
